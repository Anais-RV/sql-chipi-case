🚀 CÓMO EJECUTAR
═════════════════════════════════════════════════════════════════════

PASO 1: Configurar PostgreSQL
──────────────────────────────────────────────────────────────
En PowerShell:
  $env:PGPASSWORD="postgres"

PASO 2: Ejecutar Setup
──────────────────────────────────────────────────────────────
En PowerShell (en la carpeta del proyecto):
  make setup

Esto ejecuta:
  1. Crea base de datos chipi_case
  2. Ejecuta db/00_schema.sql
  3. Ejecuta db/01_seed.sql

PASO 3: Acceder a los datos
──────────────────────────────────────────────────────────────
Opción A - DBeaver (recomendado):
  • Abre DBeaver
  • New Database Connection → PostgreSQL
  • Host: localhost, Port: 5432
  • Database: chipi_case
  • User: postgres, Password: postgres
  • Click Finish

Opción B - psql:
  $env:PGPASSWORD="postgres"
  psql -h localhost -U postgres -d chipi_case

PASO 4: Comenzar retos
──────────────────────────────────────────────────────────────
  • Abre: challenges/day1_select_basics.md
  • Resuelve los 5 retos
  • ¡Descubre al culpable!

❓ TROUBLESHOOTING RÁPIDO
═════════════════════════════════════════════════════════════════════

Error: "FATAL: role 'postgres' does not exist"
  → Instala PostgreSQL desde postgresql.org

Error: "psql: command not found"
  → Añade C:\Program Files\PostgreSQL\16\bin a PATH

Error: "password authentication failed"
  → Configura: $env:PGPASSWORD="postgres"

Error: "database chipi_case already exists"
  → Ejecuta: make reset

Error: "Permission denied" en Makefile
  → Usa PowerShell como administrador
